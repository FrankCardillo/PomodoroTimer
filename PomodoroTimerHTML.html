Diana's Pomodoro Timer

  • 1. Use the + and - buttons to set your break and work session lengths.
  • 2. Press either the work or break button (with the number on it) to set the timer to that length.
  • 3. Press the start button to start the timer.
  • 4. Once the timer has counted down to 0, begin the process again (step 1 is optional here).
